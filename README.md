# Microsoft-Excel-Project

## Project Overview
This project focuses on cleaning and organizing a messy real world dataset using Microsoft Excel. The main objective is to apply practical data cleaning and transformation techniques to prepare raw data for accurate analysis and reporting.
The project demonstrates core Excel data analytics skills, including data importation, transformation, validation, and error correction using built in Excel tools and Power Query.

## Project Objectives
To clean and structure a messy dataset
To remove duplicates and correct data inconsistencies
To apply ETL processes using Power Query
To prepare a validated dataset suitable for analysis

## Tools Used
Microsoft Excel
Power Query Editor

## Data Cleaning Process
The following steps were carried out in completing the project:
Imported the raw dataset into a blank Excel workbook
Transformed the dataset using the Power Query Editor (ETL process)
Auto fitted rows and columns for proper data visibility
Removed duplicate entries using the Find and Replace functionality
Trimmed extra spaces from cells using formatting options
Converted the dataset into a structured Excel table
Created a Total Price column using a custom column
Conducted final checks to ensure there were no remaining errors
