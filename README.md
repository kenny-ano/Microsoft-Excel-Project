## Project Overview

This project focuses on cleaning and organizing a messy real world dataset using Microsoft Excel.

The main objective is to apply practical data cleaning and transformation techniques to prepare raw data for accurate analysis and reporting.

The project demonstrates core Excel data analytics skills, including data importation, transformation, validation, and error correction using built in Excel tools and Power Query.

---

## Project Objectives

- To clean and structure a messy dataset  
- To remove duplicates and correct data inconsistencies  
- To apply ETL processes using Power Query  
- To prepare a validated dataset suitable for analysis  

---

## Tools Used

- Microsoft Excel  
- Power Query Editor  

---

## Data Cleaning Process

The following steps were carried out in completing the project:

1. Imported the raw dataset into a blank Excel workbook  
2. Transformed the dataset using the Power Query Editor (ETL process)  
3. Auto fitted rows and columns for proper data visibility  
4. Removed duplicate entries using the Find and Replace functionality  
5. Trimmed extra spaces from cells using formatting options  
6. Converted the dataset into a structured Excel table  
7. Created a Total Price column using a custom column  
8. Conducted final checks to ensure there were no remaining errors
